<h2 id="about">🚀 About The Project</h2>

<p>This repository hosts a virtual reality application designed to transform educational paradigms. It creates an immersive active learning experience tailored for IBM SkillsBuild. By integrating advanced Local Large Language Models natively within the XR environment, it pushes the boundaries of spatial computing and interactive education.</p>

<h2 id="features">✨ Key Features</h2>

<table>
  <tr>
    <td align="center" width="150">🤖<br><b>Intelligent Guidance</b></td>
    <td>A virtual robot assistant powered by local AI helps you navigate the basics organically. Ask questions, get context aware answers, and learn interactively.</td>

<h2 id="getting_started">⚙️ Getting Started</h2>

<p>To experience this project locally, clone the repository and open it within the Unity Editor.</p>

<ol>
  <li>Ensure you have the XR Interaction Toolkit and necessary Meta Quest packages installed.</li>
  <li>Open the main scene in your Unity environment.</li>
  <li>Build and deploy to your headset.</li>
</ol>
